## QA Checklist

### Browsers
- Chrome (desktop + Android)
- Safari (iOS)
- Firefox (desktop + Android)
- Edge (desktop)

### Responsive breakpoints
- 320px (small)
- 375px (iPhone baseline)
- 414px (large phone)
- 768px (tablet)
- Desktop (≥1024px)

### Visual checks
- Settings hero strip: “Pension allowance” label does not wrap; spacing matches other stats.
- Diagnostics: hero shows the turtle icon; Back button sits below hero and above the first section.
- Budget: toolbar uses “text-left / buttons-right”; no overflow at 320px.
- Buttons: touch targets stay ≥44px on mobile; no stretched/tall pills.

### Accessibility
- Keyboard: Back button and toolbar actions are reachable by Tab.
- Focus states: visible for links/buttons.
- No text is forced into vertical letter stacking on small screens.
